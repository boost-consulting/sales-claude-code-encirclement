# 質問テクニック詳細

「無敗営業」の核心となる3つの質問テクニックの詳細解説。

---

## 1. 接戦状況を問う質問

**使用タイミング**: 案件発生時〜提案後

**目的**: 案件が接戦かどうかを見極め、注力すべきか判断する

### 具体的な質問フレーズ

```
「今回のお話、御社の中ではどのような状況でしょうか？」
「他社さんとも比較されていますか？」
「社内ではどのくらいの温度感ですか？」
「今回の件、いつ頃までにお決めになる予定ですか？」
```

### 回答パターン別の対応

| 回答 | 解釈 | 対応 |
|------|------|------|
| 「まだ何も決まっていない」 | 接戦（先延ばしパターン）の可能性 | 今やる必要性を訴求 |
| 「他社も見ています」 | 接戦（コンペパターン） | 差別化を強調 |
| 「ほぼ御社で決まり」 | 楽勝 | 効率的に進める |
| 「正直、〇〇社が有力」 | 惨敗の可能性 | 撤退か逆転策を検討 |
| 「まず内部でできるか検討中」 | 接戦（内製パターン） | 外注メリットを訴求 |

## 2. 決定の場面を問う質問

**使用タイミング**: 結果判明直後（受注・失注どちらでも）

**目的**: 「理由」ではなく「場面」を聞くことで、本当の決定要因を引き出す

### なぜ「理由」ではなく「場面」なのか

- 「理由」を聞くと、建前の回答が返ってくる
- 「場面」を聞くと、具体的な状況が語られる
- 場面から、本当の決定要因が見えてくる

### 具体的な質問フレーズ

```
「最終的にどのような場面でお決めになりましたか？」
「決定された時のことを教えていただけますか？どんな会議でしたか？」
「〇〇さんが決定された瞬間のことを、もう少し詳しくお聞かせいただけますか？」
「最終的に誰が、なんと言って決まりましたか？」
```

### 深掘りの仕方

**相手の回答例**: 「部長会議で決まりました」

**深掘り質問**:
```
「その会議では、どのようなやり取りがありましたか？」
「どなたの発言が決め手になりましたか？」
「他の方はどのような反応でしたか？」
```

### 得られた情報の活用

- **受注の場合**: 成功パターンを再現可能にする
- **失注の場合**: 敗因を特定し、次回に活かす

## 3. 裏にある背景を問う質問

**使用タイミング**: 商談中随時

**目的**: 表面的なニーズの裏にある本当の課題・動機を引き出す

### 3つのアプローチ

### A. 枕詞（まくらことば）アプローチ

相手が答えやすくなる前置きをつける。

```
「差し支えなければ、〇〇について教えていただけますか？」
「もしお分かりでしたら、〇〇は...」
「個人的なご意見で構いませんので、〇〇についてどう思われますか？」
「率直なところ、〇〇はいかがでしょうか？」
```

### B. 深掘りアプローチ

相手の発言を受けて、さらに深く掘り下げる。

**相手**: 「コスト削減が課題です」

```
「コスト削減というと、具体的にはどの部分のコストでしょうか？」
「そのコストが問題になり始めたのは、いつ頃からですか？」
「コストが削減できたら、次に何をされたいですか？」
「コスト削減を重視されている背景には、何かあるのでしょうか？」
```

### C. 特定アプローチ

選択肢を示して、相手の状況を特定する。

```
「〇〇ということは、AとBのどちらに近い状況でしょうか？」
「今のお話は、△△が原因と考えてよろしいですか？」
「つまり、□□が解決できれば、今回の課題はクリアということでしょうか？」
```

## 質問の組み合わせ例

### 例1: 初回商談

1. **接戦状況を問う**: 「今回のお話、御社の中ではどのような状況でしょうか？」
2. **背景を問う（枕詞）**: 「差し支えなければ、今回検討を始められたきっかけを教えていただけますか？」
3. **背景を問う（深掘り）**: 「〇〇が課題とのことですが、いつ頃から気になり始めましたか？」

### 例2: 提案後フォロー

1. **接戦状況を問う**: 「先日のご提案、社内ではどのような反応でしたか？」
2. **背景を問う（特定）**: 「懸念点があるとしたら、価格面と機能面、どちらでしょうか？」
3. **背景を問う（深掘り）**: 「その懸念を解消できたら、前に進めそうですか？」

### 例3: 失注後ヒアリング

1. **決定の場面を問う**: 「最終的にどのような場面でお決めになりましたか？」
2. **深掘り**: 「その会議で、決め手になった発言は何でしたか？」
3. **背景を問う**: 「もし当社でもう一度提案するとしたら、何を変えるべきでしょうか？」

---

## 質問時の注意点

1. **一度に多く聞かない** - 1つ聞いたら、相手の回答を待つ
2. **沈黙を恐れない** - 相手が考える時間を与える
3. **メモを取る** - 相手の発言を大切にしている姿勢を見せる
4. **オウム返しを活用** - 「〇〇ということですね」で確認
5. **ジャッジしない** - 相手の回答を否定しない
