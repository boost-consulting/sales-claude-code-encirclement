# 質問テクニック詳細

「無敗営業」第2章・第4章に基づく質問テクニックの詳細解説。
接戦を制する3つの質問と、質問力の4つのフェーズをカバー。

---

## 1. 接戦状況を問う質問

**使用タイミング**: 案件発生時〜提案後

**目的**: 案件が接戦かどうかを見極め、注力すべきか判断する

### 具体的な質問フレーズ

```
「今回のお話、御社の中ではどのような状況でしょうか？」
「他社さんとも比較されていますか？」
「社内ではどのくらいの温度感ですか？」
「今回の件、いつ頃までにお決めになる予定ですか？」
```

### 回答パターン別の対応

| 回答 | 解釈 | 対応 |
|------|------|------|
| 「まだ何も決まっていない」 | 接戦（先延ばしパターン）の可能性 | 今やる必要性を訴求 |
| 「他社も見ています」 | 接戦（コンペパターン） | 差別化を強調 |
| 「ほぼ御社で決まり」 | 楽勝 | 効率的に進める |
| 「正直、〇〇社が有力」 | 惨敗の可能性 | 撤退か逆転策を検討 |
| 「まず内部でできるか検討中」 | 接戦（内製パターン） | 外注メリットを訴求 |

---

## 2. 決定の場面を問う質問

**使用タイミング**: 結果判明直後（受注・失注どちらでも）

**目的**: 「理由」ではなく「場面」を聞くことで、本当の決定要因を引き出す

### なぜ「理由」ではなく「場面」なのか

- 「理由」を聞くと、建前の回答が返ってくる
- 「場面」を聞くと、具体的な状況が語られる
- 場面から、本当の決定要因が見えてくる

### 具体的な質問フレーズ

```
「最終的にどのような場面でお決めになりましたか？」
「決定された時のことを教えていただけますか？どんな会議でしたか？」
「〇〇さんが決定された瞬間のことを、もう少し詳しくお聞かせいただけますか？」
「最終的に誰が、なんと言って決まりましたか？」
```

### 深掘りの仕方

**相手の回答例**: 「部長会議で決まりました」

**深掘り質問**:
```
「その会議では、どのようなやり取りがありましたか？」
「どなたの発言が決め手になりましたか？」
「他の方はどのような反応でしたか？」
```

### 得られた情報の活用

- **受注の場合**: 成功パターンを再現可能にする
- **失注の場合**: 敗因を特定し、次回に活かす

---

## 3. 裏にある背景を問う質問

**使用タイミング**: 商談中随時

**目的**: 表面的なニーズの裏にある本当の課題・動機を引き出す

### 3つのアプローチ

### A. 枕詞（まくらことば）アプローチ

相手が答えやすくなる前置きをつける。

```
「差し支えなければ、〇〇について教えていただけますか？」
「もしお分かりでしたら、〇〇は...」
「個人的なご意見で構いませんので、〇〇についてどう思われますか？」
「率直なところ、〇〇はいかがでしょうか？」
```

### B. 深掘りアプローチ

相手の発言を受けて、さらに深く掘り下げる。

**相手**: 「コスト削減が課題です」

```
「コスト削減というと、具体的にはどの部分のコストでしょうか？」
「そのコストが問題になり始めたのは、いつ頃からですか？」
「コストが削減できたら、次に何をされたいですか？」
「コスト削減を重視されている背景には、何かあるのでしょうか？」
```

### C. 特定アプローチ

選択肢を示して、相手の状況を特定する。

```
「〇〇ということは、AとBのどちらに近い状況でしょうか？」
「今のお話は、△△が原因と考えてよろしいですか？」
「つまり、□□が解決できれば、今回の課題はクリアということでしょうか？」
```

---

## 質問の組み合わせ例

### 例1: 初回商談

1. **接戦状況を問う**: 「今回のお話、御社の中ではどのような状況でしょうか？」
2. **背景を問う（枕詞）**: 「差し支えなければ、今回検討を始められたきっかけを教えていただけますか？」
3. **背景を問う（深掘り）**: 「〇〇が課題とのことですが、いつ頃から気になり始めましたか？」

### 例2: 提案後フォロー

1. **接戦状況を問う**: 「先日のご提案、社内ではどのような反応でしたか？」
2. **背景を問う（特定）**: 「懸念点があるとしたら、価格面と機能面、どちらでしょうか？」
3. **背景を問う（深掘り）**: 「その懸念を解消できたら、前に進めそうですか？」

### 例3: 失注後ヒアリング

1. **決定の場面を問う**: 「最終的にどのような場面でお決めになりましたか？」
2. **深掘り**: 「その会議で、決め手になった発言は何でしたか？」
3. **背景を問う**: 「もし当社でもう一度提案するとしたら、何を変えるべきでしょうか？」

---

## 質問時の注意点

1. **一度に多く聞かない** - 1つ聞いたら、相手の回答を待つ
2. **沈黙を恐れない** - 相手が考える時間を与える
3. **メモを取る** - 相手の発言を大切にしている姿勢を見せる
4. **オウム返しを活用** - 「〇〇ということですね」で確認
5. **ジャッジしない** - 相手の回答を否定しない

# 質問力の4つのフェーズ（第4章）

品定め気味で警戒心の強い顧客から情報を引き出すためのフレームワーク。
**質問力が特に必要なのは「接戦案件」**。楽勝や惨敗の案件では発揮する機会が少ない。

---

## フェーズ概要

| フェーズ | 内容 | ポイント | よくある失敗 |
|----------|------|----------|--------------|
| **①土台作り** | 雰囲気づくり | つかみとアイスブレイク、双方向な説明 | ノンストップで話し続けてしまう |
| **②切り込む「聞く」** | 質問への対応 | お客さま質問への回答、枕詞の活用 | 質問の意図を確認せずずれた回答 |
| **③深掘りする「聴く」** | 傾聴と深掘り | 深掘り質問、積極的傾聴 | 顧客の話を途中でさえぎる |
| **④具体化する「訊く」** | 核心に迫る | 特定質問、核心質問 | オープン/クローズド質問が一辺倒 |

---

## ①土台作り

**質問力の前提** - お客さまが親切に教えてくださる関係性を構築する

### つかみとアイスブレイクの3つのポイント

1. **信頼感を与える立ち居振る舞い** - ビジネスマナーの基本
2. **自分から自己開示する** - 簡単なバックグラウンドを話す
3. **相手に対する興味・関心を伝える** - 公開情報を話題にする

**具体例:**
```
「先日、新聞に出ていた御社の新商品発表の記事ですが...」
「名刺に記載されているこちらの部署では、どんなことを...」
```

### 双方向な説明

- 一方的に話さず、相手の表情・目線に注目
- タイミングを見て理解度を確認
- お客さまがページをめくる動きにペースを合わせる

**NG例:**
- 会ってすぐ一方的に話し始める
- 自社パンフレットをそのまま読む

---

## ②切り込む「聞く」

### お客さまからの質問への対応

**重要**: こちらが質問する前に「きちんと話が通じる相手」と思っていただく

- 質問の意図を確認してから回答
- 回答後、趣旨を満たせているか確認
- 自分の聞きたいことだけ聞いて、相手の質問をおろそかにしない

### 枕詞（まくらことば）の3種類

質問しづらいことを聞きやすくする「前置き」

#### 1. 状況を説明する枕詞

```
「前にもお伺いしたことかもしれませんが…」
「すでにご存じかとは存じますが…」
「先日○○とおっしゃっていた点が気になっておりまして…」
```

#### 2. 前提を変更する枕詞（法人営業で特にお勧め）

```
「もし仮に○○という点がクリアされたら…」
「もし仮にわがままを全部言えるとしたら…」
「あくまで○○さんの個人的なご意見で構いませんので…」
```

#### 3. 質問理由を伝える枕詞

```
「○○の準備をさせていただくために、1つだけ質問させていただいてもよろしいでしょうか…」
「いただいたお時間を無駄にしないために…」
「私から一方的にお話しすぎてはいけないので…」
```

### 枕詞の活用例

**競合状況を聞く場合:**
- ❌「当社は今、何番目につけていますか」（不躾）
- ✅「あくまでも山田さんの個人的なご意見で構いませんので、当社は何番目くらいだと思われますか」

**予算を聞く場合:**
- ❌「ご予算はいくらですか」（直球すぎる）
- ✅「今回の提案ではいくつかの案をご提示したいので、ご予算の上限はどれくらいでしょうか」

---

## ③深掘りする「聴く」

### ピラミッド構造を意識した4つの深掘り質問

| # | 内容 | 具体例 | 方向 |
|---|------|--------|------|
| 1 | 発言内容を**明確にする**質問 | 「と、おっしゃいますと？」 | 言い換え |
| 2 | **詳細を引き出す**質問 | 「具体的には？」 | 下へ↓ |
| 3 | **背景を引き出す**質問 | 「なぜでしょうか？」 | 上へ↑ |
| 4 | **網羅感を確認**する質問 | 「他にはありますか？」 | 横へ↔ |

**ポイント**: 短い一言で相手にボールを戻す

### 積極的傾聴（アクティブリスニング）

- 相槌やアイコンタクト
- メモをとる動き
- お客さまの話を途中でさえぎらない
- 「なるほど」と言い換えたり要約する

**効果**: 「この営業はわかってくれている」と感じてもらえる

**沈黙への対処**: お客さまは頭の中で考えを整理中かもしれない。沈黙を恐れない。

---

## ④具体化する「訊く」

### 質問の4つのタイプ（広げる→絞る）

| # | タイプ | 定義 | 具体例 |
|---|--------|------|--------|
| 1 | **オープン** | 自由に答えていただく | 「御社の課題についてはどのようにお考えですか?」 |
| 2 | **条件付きオープン** | 制約条件をつけて自由に | 「御社の課題について、**特にここ一か月**議論されているものは?」 |
| 3 | **選択肢付きクローズド** | 例示で選択肢を示す | 「課題の種類は?**例えば**ビジョン、マネジメント層のリーダーシップなど…」 |
| 4 | **クローズド** | 回答範囲を限定 | 「御社の課題はXXXと感じましたが、**合っていますか?**」 |

**ポイント**: 「条件付きオープン」と「選択肢付きクローズド」を駆使できれば、大体の情報は聞き出せる

### 核心質問

「そもそも」の部分を聞き出す質問。ビジョン・目標、達成手段に焦点を当てる。

**ビジョン・目標に関する例:**
```
「そもそも、なぜそういう目標を設定されたのでしょうか?」
「そのビジョン実現に対して、"うまくいっていること""思うように進まないこと"は?」
```

**達成手段に関する例:**
```
「なぜ、(一年後ではなく)今のタイミングなのでしょうか?」
「そのまま自社内で進められるのではまずいのでしょうか?」
「なぜ、現在契約されている他社だと不十分なのでしょうか?」
```

**リプレイス案件で特に有効:**
```
「すでに他社さんとお付き合いされていて、御社のことを十分に理解されている会社があるはずなのに、初めてお会いした弊社にも提案の機会をいただけるのは、なぜでしょうか」
```

# 課題解決質問（応用レベル）

お客さまの課題をともに解決するパートナーとして、議論や対話を展開する技術。
**習得は難しいが、使えるようになると飛躍的に受注率が高まる**。

---

## 課題解決質問の4ステップ

| ステップ | 質問の狙い | 質問の言い回し（例） |
|----------|-----------|---------------------|
| **①現状把握** | 状況や背景を把握 | 「貴社の活動について、いくつかお伺いしてもよろしいでしょうか?」 |
| **②深掘り** | 現状への不満の原因を追求 | 「今おっしゃった『〜したい』について、もう少し詳しくお伺えますか?」 |
| **③気づかせ** | 問題の重要度を明らかにする | 「●●ができないとどのような問題が起こるのでしょうか?」 |
| **④つなぎ** | 解決策（自社提案）へスムーズに | 「何が解決されると、貴社の成功に一番近づくのでしょうか?」 |

---

## ①現状把握の質問

**始め方の例:**
```
「最近は、何か、お困りごとはありませんか」
「今回の案件の背景について、ちょっと聞かせていただけませんか」
「最近、御社は中期経営計画を発表されましたが、それについて少し伺ってもよろしいですか」
```

**ポイント**: 現状への不満が見え隠れしたら、深掘りのチャンス

---

## ②深掘り質問

**タイミング**: お客さまが「不満」「不安」「懸念」を漏らした瞬間

**会話例:**
```
営業: 「この間、御社は三か年計画を発表されましたが、どんな感じなんですか」
お客さま: 「いや、それが......まとまらなくて、大変で......」
営業: 「今、『まとまらない』とおっしゃいましたが、もう少し詳しくお伺いできますか」
```

---

## ③気づかせ質問

**目的**: 問題が「放っておけない、重要度の高い問題」と認識してもらう

**コツ**: 視点を「現在」から「未来」に移す

**質問例:**
```
「その3億円のギャップについて、どう実現するか、ずっとまとまらない状態が続いてしまったら、いったいどんなことが起こりますか」
「もし、この現状が続くとどのような影響が御社にとって出てくるのでしょうか?」
```

**枕詞「もし仮に......」が特に有効** - 話題を未来に展開しやすくなる

---

## ④つなぎ質問

**目的**: 当社の提案でどんなインパクトがあるかを、お客さま自身の言葉で語っていただく

**質問例:**
```
「もし、弊社とディスカッションさせていただきながら、その3億円のギャップを埋める戦略が明確になったら、どんなインパクトがあるでしょうか」
「ある年齢層に狙いを定めてリピート率を高めることができたら、どのようなインパクトが御社にとってはありますか?」
```

---

## 課題解決質問の会話イメージ

| ステップ | 営業 | 顧客 |
|----------|------|------|
| **現状把握** | 「御社で特に力を入れてターゲットにされているのはどのような年代のお客さまでしょうか?」 | 「本当は、もっとターゲットやコンセプトについて考えた方がいいのかもしれないけど…」 |
| **深掘り** | 「『本当はもっと考えた方がいい』について、もう少し詳しく伺えますか?」 | 「最近、リピート率が下がっていて、何かテコ入れしたいとは思っているんですよね」 |
| **気づかせ** | 「もし、この現状が続くとどのような影響が御社にとって出てくるのでしょうか?」 | 「リピート率が下がると広告費用もかさむし、利益率がだいぶ圧迫されてしまいますね」 |
| **つなぎ** | 「リピート率を高めることができたら、どのようなインパクトがありますか?」 | 「やることがハッキリするのですごくいいですね!」 |

**効果**: 重要な発言がお客さまから出てくる → 一方的な売り込みにならない → 受注率が格段に上がる

## 第4章調査結果: お客さまが聞いてほしいこと

| 順位 | 項目 | 割合 |
|------|------|------|
| 1位 | 会社が求めていることや目指している方向性 | 約58% |
| 2位 | 会社が困っていることや悩んでいること | 約52% |
| 3位 | 会社のビジネスの内容 | 約32% |
| 4位 | 案件の予算に関わること | 約25% |
| 5位 | 発注先を決める際の判断基準 | 約22% |

**重要な示唆**: お客さまが営業に聞いてほしいのは「方向性」と「悩み」という**上流の情報**。
予算やスケジュールより、**本質的なニーズ**を把握してほしいと思っている。

**50.8%がヒアリング不足で営業に失望**している。
