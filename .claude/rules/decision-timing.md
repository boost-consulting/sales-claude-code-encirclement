# 決定タイミングと本命 vs 当て馬

「無敗営業」第3章より。お客さまが「決める瞬間」の真実と、
ハイパフォーマー vs ローパフォーマーの違いを解説。

---

## 核心メッセージ

> **ほとんどの商談は、自分の想像と違う場面で決着している**

---

## お客さまが決める瞬間（逆順）

時間の流れと**逆順**に並べると、決定タイミングの真実が見える：

```
┌─────────────────────────────────────┐
│ 商品・サービスを実際に見て、       │  ← 営業が想像する決定タイミング
│ 見積もりを確認して                 │
└─────────────────────────────────────┘
                ▲
┌─────────────────────────────────────┐
│ 商品・サービスに関する説明を聞いて │
└─────────────────────────────────────┘
                ▲
┌─────────────────────────────────────┐
│ ★ 営業の行動を見て ★              │  ← 実際の決定タイミング
└─────────────────────────────────────┘
                ▲
┌─────────────────────────────────────┐
│ 営業もしくは会社の名前で           │  ← 楽勝案件
└─────────────────────────────────────┘
```

### 重要な示唆

| 営業が思う決め手 | 実際の決め手 |
|-----------------|-------------|
| 見積もり・価格 | **営業の行動** |
| 商品説明 | **第一印象・姿勢** |
| 提案内容 | **会社の評判** |

**→ 決定は「意外と手前」で起きている**

---

## 各決定タイミングの特徴

### レベル1: 見積もり確認後に決める（最も遅い）

**特徴:**
- 説明も頑張らなければいけない
- 工数がかかる
- 「試しに使ってみてから」と言われる
- 提案期間が長くなる

**リスク:**
- 「迷った末に価格で決める」パターンに陥りやすい
- 特別値引きをしてしまう罠にはまる
- 営業コストが膨大になる

### レベル2: 商品説明を聞いて決める

**特徴:**
- 実際に使ってみなくても決めてくれる
- 見積もり金額を確認しなくても「いいね」となる

**メリット:**
- 提案書作成に膨大な時間をかけなくて済む
- 本来しなくてもいい値引きを避けられる

### レベル3: 営業の行動を見て決める

**特徴:**
- 提案前の姿勢・行動で信頼を獲得
- 「この人にお願いしたい」と決まる

**事例:**
> E社長：「熱心にメモをとる姿勢を見て、心のなかで決めていた」

### レベル4: 会社の名前で決める（楽勝案件）

**特徴:**
- 信頼・ブランドで即決
- 接戦に強くなることで、このゾーンが広がる

---

## 核心ルール

> **決定タイミングが遅くなればなるほど、営業コストが増える**

```
決定タイミング    営業コスト
    遅い    →    高い（価格競争に巻き込まれる）
    早い    →    低い（信頼で決めてもらえる）
```

**目標**: お客さまから信頼してもらって「手前」で決めていただく

---

## 2種類の営業

| 上流で決めている営業 | 戦う前に負けている営業 |
|---------------------|----------------------|
| 行動・姿勢で信頼を獲得 | 価格・提案書で勝負 |
| 営業コスト低い | 営業コスト高い |
| 楽勝ゾーンが広がる | 接戦・惨敗が多い |

### ハイパフォーマーの特徴

- 早い段階で「この担当者なら間違いない」と思われる
- 商品・サービスの説明**より前**に信頼を得る
- 提案を出す前の**上流段階**で決着をつける
- **大量の楽勝案件**で売上を組み立てる

### ローパフォーマーの特徴

- 信頼を得られないまま提案段階に進む
- 価格競争に巻き込まれる
- **惨敗案件を多く抱える**

---

## 本命 vs 当て馬の見分け方

### 本命（選ばれる営業）への対応

| 段階 | 顧客の対応 |
|------|----------|
| ヒアリング | 「初期段階から一緒に議論させてください」 |
| プレゼン | 「上司に同席させますのでプレゼンお願いします」 |
| 見積提出 | 「他社はこの価格です」（情報開示） |
| 結果連絡 | 「お願いします」 |

### 当て馬（比較用の営業）への対応

| 段階 | 顧客の対応 |
|------|----------|
| 提案依頼 | 「1週間で提案ください」（短納期） |
| 提案書作成 | 「忙しいので提案はメールでお願いします」 |
| 見積提出 | 「他社の価格は秘密」（情報非開示） |
| 結果連絡 | 「惜しくも今回は…」 |

### 見分けるポイント

**本命のサイン:**
- 初期から一緒に議論できる
- 決裁者に会わせてもらえる
- 競合の情報を教えてもらえる

**当て馬のサイン:**
- 短納期で提案を求められる
- 対面の機会が少ない（メールで済まされる）
- 情報が出てこない

## 当て馬案件への対処

### よくある声（成果が出ない営業）

> 「いつもお客さまが『すぐに提案をくれ』とおっしゃるんです」

**真実**: これは「すでに決着がついている」ケース
- 他社に決まっている案件の提案を一生懸命している
- 当て馬案件を多く抱えている状態

### 苦しい事態を防ぐ4ステップ

1. **接戦状況を確認する**
   - 多くが惨敗案件であることに気づく
   - 接戦を見極めることが必要

2. **リソースを集中する**
   - 惨敗案件にかける時間を減らす
   - 限られた接戦案件にリソースを集中

3. **チームで学習する**
   - 「決定の場面を問う質問」をチーム単位で実践
   - 他メンバーの決定場面から学ぶ

4. **背景を深掘りする**
   - 勝負どころの接戦案件で「裏にある背景を問う質問」を活用
   - 限られた勝機に対してベストな努力を続ける

---

## 手前で決めてもらうための条件

> **受注・失注後のヒアリングで、決定場面に関する事実を日頃からたくさん聞いておく**

- 具体的な情報をつかんでいるからこそ「この行動をお客さまは見ている」と確信できる
- 確信しながら営業活動ができる

## 事例: E社長の決定場面

### 競合4社の比較

| 営業 | アプローチ | 結果 |
|------|----------|------|
| 1社目 | 「御社の組織体制はまだまだ脆弱です」 | 気分を害された |
| 2社目 | 「ここが弱点です」「ここも課題です」 | プライドを傷つけられ不機嫌に |
| 3社目 | タイミング遅れ | 不利な状況 |
| **4社目（著者）** | 背筋を伸ばし、素直に話を聞き、ノートを取り始めた | **受注** |

### E社長の決定要因

> 「熱心にメモをとっている姿を見て、心のなかでは決めていたんです」

**教訓:**
- 価格ではなかった
- 提案前に決まっていた
- 決定的な要因は「傾聴の姿勢」

---

## 決定場面から得た学び

### 上位役職者への対応

- 「課題」「弱み」といったニュアンスの言葉は、表現や伝え方に注意

### 経営者相手の商談

- 提案書やプレゼンだけでなく、**細かい所作までチェック**される

### オーナー社長相手の提案

1. まず、しっかりと**傾聴**する
2. 心血を注いで経営されてきたことに**敬意を表す**
3. 自社からの提案は、**その後にすべき**
