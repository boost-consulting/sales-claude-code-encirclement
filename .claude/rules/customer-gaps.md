# 4つの不満と4つの力

「無敗営業」第3章より。お客さまの不満を解消し、
「わかってくれる営業」になるためのフレームワーク。

---

## 最凶のライバルは、過去の「ガッカリ営業」

### 核心

競合他社ではなく、**お客さまが過去に経験した「ガッカリ営業」**が最凶のライバル。
その記憶が、営業への不信感・警戒心を生んでいる。

### 重要データ

> お客さまが「わかってくれる」と感じる営業は**6人に1人**

つまり、6人中5人は「ズレた営業」「ガッカリ営業」と思われている。

---

## 顧客の防御反応（ガッカリ営業への対策）

| 顧客の発言 | 本音 |
|-----------|------|
| 「すぐに提案くれませんか?」 | 深く関わりたくない |
| 「とりあえずいくら?」 | 比較材料だけほしい |
| 「上司には会わせません」 | 信頼できるか分からない |
| 「もっと安くなりませんか?」 | 価値を感じていない |

### お客さまの心理

> 「どうせ、6分の5のハズレの方だろうな」

- まだ関係が構築できていない段階では、**負の予測**で接する
- 拒絶的・否定的な態度が**無意識のうちに**身についている

---

## 顧客の「冷たい反応」の本当の理由

### 上司に会わせてもらえない本当の理由

**あなたの力不足ではないかもしれない**

顧客の過去の経験：
> かつて他社のハズレ営業を上司に会わせてしまい、「お前のところで処理しておけ！」と叱られた

→ 同じ目に遭いたくないから、面会依頼を断っている

### 「期待はずれ」と言われる本当の理由

**あなたの提案レベルが低いからではないかもしれない**

顧客の過去の経験：
- 数々の営業担当者から「期待はずれの提案書」を見せられてきた
- だから最初から期待値が低い

### 顧客の心理メカニズム

```
過去のガッカリ体験
    ↓
「この人もハズレだろう」と予測
    ↓
厳しいコミュニケーション（防御反応）
    ↓
あなたが「ズレた営業」と同じ行動をすると地雷が爆発
```

---

## 4つの不満（調査結果）

**Q: 営業担当者に失望して発注を控えるとしたら、その理由は？**
（n=309、2017年11月 TORiX調べ）

| 順位 | 不満の内容 | 割合 |
|-----|----------|------|
| 1位 | 要件ヒアリングが不十分、情報の把握ができていない | **50.8%** |
| 2位 | 営業担当者としての魅力や価値を感じない | **43.7%** |
| 2位 | 営業担当者としての動きが悪い | **43.7%** |
| 4位 | 顧客の意図に沿わない提案を出してくる | 約37% |

**重要な示唆:**
> 「提案内容が悪い」（37%）よりも、**ヒアリング・人柄・動き**への不満が圧倒的に多い

→ 提案の中身より、**営業としての姿勢・行動**が問われている

## 4つの力（ズレを解消するスキル）

| ズレの種類 | 解消する力 |
|-----------|----------|
| ヒアリング不足 | **質問力** |
| 魅力・価値を感じない | **価値訴求力** |
| 動きが悪い | **提案行動力** |
| 意図に沿わない提案 | **提案ロジック構築力** |

### 四本柱モデル

```
                ┌─────────────┐
                │ 提案ロジック │
                │   構築力     │
                └──────┬──────┘
                      ╱ ╲
                     ╱   ╲
          ┌─────────┐     ┌─────────┐
          │  質問力  │     │価値訴求力│
          └────┬────┘     └────┬────┘
                ╲             ╱
                 ╲           ╱
                ┌──────┴──────┐
                │  提案行動力  │
                └─────────────┘
```

---

## 各力の詳細

### 1. 質問力

- 顧客の状況・ニーズを正確に把握
- 「接戦を制する3つの質問」はこの一部
- お客さまとの**情報ギャップを埋める**

**注意点:**
> ただ質問ばかりしていても、うまく進まない場合がある

理由: お客さまは、こちらが聞いたことに対して**必ず答えてくれるわけではない**

### 2. 価値訴求力

- 営業としての魅力・価値を伝える
- 「答えようかな」と思っていただける**価値を提供**する
- 質問力とは**コインの表と裏**の関係

```
質問力 ←→ 価値訴求力
（聞く）   （提供する）
```

### 3. 提案ロジック構築力

- 顧客の意図に沿った提案を構築
- 接戦案件で「当社を選ぶ理由」を作れる
- 営業の一方的な都合ではなく、**お客さまの立場から**選ぶべき理由を提示

### 4. 提案行動力

- 迅速で適切なアクション
- 一つひとつの動作にかかる時間が短く済む
- 「壁」に突き当たったとき、適切なアクションが取れる
- 営業活動の**生産性が上がる**

---

## 4つの力の関係性

### ハイパフォーマーの秘訣

> **4つの力をぐるぐる回している**

```
質問力の発揮（聞く）
    ↓↑
価値訴求力の発揮（役立つ）
    ↓↑
情報ギャップが解消される
    ↓
関係が深まる
    ↓
提案ロジック構築力で設計
    ↓
提案行動力で実行
```

---

## ガッカリ営業の特徴（避けるべき行動）

```
┌─────────────────┐
│ 一方的な提案     │
│ 連絡がこない     │
│ ヒアリング不足   │  →  顧客の不信感を蓄積
│ 期待とのズレ     │
│ 悪いレスポンス   │
└─────────────────┘
```

### 失注パターン vs 受注パターン

| 失注パターン | 受注パターン |
|-------------|-------------|
| 上から目線で課題を指摘 | 素直に話を聞く |
| 一方的に提案をまくしたてる | ノートを取りながら傾聴 |
| プライドを傷つける | 顧客理解の姿勢を見せる |

## 地雷を踏まないために

### 解決への道筋

```
お客さまとの「ズレ」を解消するスキルが上がる
    ↓
「わかってくれる（＝ズレない）営業」に自然と近づく
    ↓
お客さまの防御反応が解除される
```

### 2つのポイント

1. **地雷を踏まない** - 「ズレた営業」とみなされる行動を避ける
2. **スキルを強化する** - 「お客さまをわかってくれる営業」のスキルを身につける

---

## 視点転換

| 営業が思う原因 | 本当の原因 |
|--------------|----------|
| 自分の力不足 | 過去の営業への不信感 |
| 提案のレベルが低い | 期待値が最初から低い |
| 競合が強い | 過去のガッカリ営業の影響 |

**結論**: 地雷の種類を理解し、「ズレた営業」と同じ行動を避けることで、この状況は回避可能
