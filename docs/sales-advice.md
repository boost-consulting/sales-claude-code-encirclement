# 営業相談コマンド

「無敗営業」フレームワークに基づいて、営業状況を分析し、具体的なアクションを提案する対話型アドバイザー。

## 実行手順

### Step 1: 状況ヒアリング

ユーザーに以下を質問して状況を把握する:

1. **案件の概要**: 「どんな案件ですか？（商品/サービス、顧客、金額感など）」
2. **現在のフェーズ**: 「今どの段階ですか？（初回接触/提案済み/クロージング中/結果判明後）」
3. **感触**: 「今の感触はどうですか？（勝てそう/五分五分/厳しい）」

### Step 2: 案件分類

ヒアリング結果から案件を分類:
- **楽勝**: 効率的に進める
- **接戦**: 注力すべき（Step 3へ）
- **惨敗**: 早期撤退を検討

### Step 3: 接戦パターン特定（接戦の場合）

以下のどれに該当するか確認:
1. **当社 or 他社**: コンペ状態 → 差別化を訴求
2. **今やるか**: 先延ばし → 今やらないリスクを提示
3. **内製か外注か**: 自社開発検討中 → 外注メリットを訴求

### Step 4: 推奨アクション提示

以下の形式で回答:

```
## 状況分析
- 案件分類: [楽勝/接戦/惨敗]
- 接戦パターン: [該当する場合]
- 現在のフェーズ: [初回/提案後/クロージング/結果判明後]

## 推奨アクション
1. [具体的なアクション]
2. [具体的なアクション]
3. [具体的なアクション]

## 使える質問・フレーズ
- 「[そのまま使える質問文]」
- 「[そのまま使える質問文]」

## 確認すべき情報（BANTCH）
- [ ] Budget: [確認状況]
- [ ] Authority: [確認状況]
- [ ] Needs: [確認状況]
- [ ] Timing: [確認状況]
- [ ] Competitor: [確認状況]
- [ ] Human: [確認状況]

## 次のステップ
[具体的な次のアクション]
```
